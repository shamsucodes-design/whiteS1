# Love Calculator

A small front-end "love calculator" web page.

## What it does
- Takes two names (your name + their name)
- Computes a deterministic "love score" (so the score stays the same for the same pair)
- Shows the result on the page

## How to use
1. Open `index.html` in a browser.
2. Enter both names.
3. Click **Calculate Love**.

## Notes
- This is a standalone static page (no backend required).
- The score is generated using a simple hash-like function based on the names, so it is consistent for the same inputs.
